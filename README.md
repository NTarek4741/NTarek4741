## Featured Project

### 🚀 [mlx-nim](https://github.com/NTarek4741/mlx-nim)

> Local LLM/VLM inference server for Apple Silicon. No cloud, no telemetry, no data leaving your machine.

| Feature | Detail |
|---|---|
| **API surfaces** | OpenAI-compatible, Anthropic-compatible, Ollama-compatible |
| **Inference** | LLM + VLM, streaming, tool calling, structured output, reasoning blocks |
| **Performance** | 2.4× faster prompt processing vs. Ollama (201 t/s vs. 85 t/s) |
| **Optimizations** | KV cache quantization, speculative decoding, in-memory model caching |
| **Clients** | Claude Code, n8n, OpenCode, curl |
